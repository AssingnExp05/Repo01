PetCare File Upload Directory

This directory contains uploaded pet photos and adoption documents.

Security:
- Only image files (jpg, jpeg, png, gif, bmp, webp) are allowed
- PHP and other executable files are blocked
- Directory listing is disabled

File Naming:
- Files are automatically renamed with unique identifiers
- Format: [unique_id]_[timestamp].[extension]

Maintenance:
- Regularly clean up old or unused files
- Monitor disk space usage
- Backup important files
